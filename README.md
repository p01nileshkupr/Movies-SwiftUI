
# Movies
- A SwiftUI app for listing top 100 IMDB movies

## Technologies Used
- Language: Swift 5.7
- Tool: XCode 14.2
- Data: JSON
- Design Patterns: Observable, State, Codable, MVVM, Async/Await, etc.
- Tests: XCUnitTest
